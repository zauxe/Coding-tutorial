What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. It controls colors, fonts, layouts, spacing, and much more!

üí° Think of CSS like...

The interior designer of your website - it takes the basic structure (HTML) and makes it look amazing!

CSS Syntax

CSS is made up of rules that target HTML elements and apply styling to them.

selector {
    property: value;
    another-property: another-value;
}

Example:

h1 {
    color: blue;
    font-size: 32px;
    text-align: center;
}
  • h1 - The selector (targets all h1 elements)
  • color: blue; - A property-value pair
  • { } - Curly braces contain all the styles

Common CSS Selectors

Element Selector

p {
    font-size: 16px;
}

Targets all <p> elements

Class Selector

.highlight {
    background-color: yellow;
}

Targets elements with class="highlight"

ID Selector

#header {
    height: 100px;
}

Targets element with id="header"

Descendant Selector

nav a {
    text-decoration: none;
}

Targets <a> elements inside <nav>

üéØ How to Target HTML Elements in CSS

Understanding CSS selectors is crucial! Here's how to target different HTML elements:

Target Classes with . (dot)

HTML:

<div class="container">Content</div>

CSS:

.container {
    max-width: 1200px;
    margin: 0 auto;
}

Target IDs with # (hash)

HTML:

<div id="header">Navigation</div>

CSS:

#header {
    background-color: #333;
    height: 80px;
}

Target Elements Directly

HTML:

<h1>Page Title</h1>
<p>Paragraph text</p>

CSS:

h1 {
    font-size: 2rem;
    color: #333;
}

p {
    line-height: 1.6;
}

Combine Selectors

HTML:

<nav class="nav-links">
    <a href="#">Home</a>
</nav>

CSS:

.nav-links a {
    text-decoration: none;
    padding: 10px;
}

.nav-links a:hover {
    background-color: #f0f0f0;
}

üí° Quick Reference

  • Classes (can be used multiple times): .classname
  • IDs (should be unique): #idname
  • Elements (target all of that type): h1, p, div
  • Nested (element inside another): .container p

üé® CSS Custom Properties (Variables)

Use CSS variables to make your stylesheets more maintainable and consistent!

Defining Variables

/* Define variables in :root (available everywhere) */
:root {
    --main-color: #3498db;
    --text-color: #333;
    --spacing: 1rem;
    --border-radius: 8px;
}

Using Variables

/* Use variables with var() */
.button {
    background-color: var(--main-color);
    color: var(--text-color);
    padding: var(--spacing);
    border-radius: var(--border-radius);
}

.card {
    border: 1px solid var(--main-color);
    padding: calc(var(--spacing) * 2);
}

Theme Switching (like this website!)

/* Light theme (default) */
:root {
    --bg-color: #ffffff;
    --text-color: #333333;
}

/* Dark theme override */
.dark {
    --bg-color: #1a1a1a;
    --text-color: #ffffff;
}

body {
    background-color: var(--bg-color);
    color: var(--text-color);
}

üåü Benefits of CSS Variables

  • Consistency: Change a color once, update everywhere
  • Theme switching: Easy light/dark mode implementation
  • Maintainability: No need to find/replace values across files
  • Calculations: Use with calc() for dynamic values

‚ùå Common CSS Mistakes to Avoid

‚ùå Missing Semicolons

.container {
    width: 100px
    height: 50px
    margin: 10px
}

‚úÖ All Semicolons Present

.container {
    width: 100px;
    height: 50px;
    margin: 10px;
}

‚ùå Forgetting the Dot for Classes

container {
    width: 100%;
}

‚úÖ Proper Class Targeting

.container {
    width: 100%;
}

#header {
    background: blue;
}

‚ùå Missing Units

.box {
    width: 100;
    margin: 10;
    font-size: 16;
}

‚úÖ Include Units

.box {
    width: 100px;
    margin: 10px;
    font-size: 16px;
}

‚ùå Overusing !important

.text {
    color: red !important;
    font-size: 20px !important;
}

‚úÖ Use Specificity Instead

.container .text {
    color: red;
    font-size: 20px;
}

Responsive Design Basics

Make your website look great on all devices using media queries:

/* Mobile First Approach */
.container {
    width: 100%;
    padding: 10px;
}

/* Tablet Styles */
@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
}

/* Desktop Styles */
@media (min-width: 1024px) {
    .container {
        width: 1000px;
        padding: 20px;
    }
}

üì± Mobile: 100% width with small padding

üìü Tablet: 750px width, centered

üíª Desktop: 1000px width with larger padding
